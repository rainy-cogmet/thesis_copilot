### 要求

作为终审环节，从全文角度进行最终评审：
1. 整体框架是否完整、合理
2. 各章之间的逻辑关系是否清晰
3. 创新点是否阐述充分
4. 研究方法是否描述清楚
5. 结论是否有足够的数据/论证支撑
6. 摘要是否准确概括全文
7. 格式规范是否统一
8. 参考文献是否完整规范

请以如下 JSON 格式回复：

{{
  "pass": true或false,
  "feedback": "全文终审意见",
  "score": 1到10的评分,
  "chapter_issues": {{
    "章节名": "该章存在的问题（如果有）"
  }}
}}

如果通过，chapter_issues 为空对象。
评分标准：8分以上为通过，7分以下需要继续修改。
